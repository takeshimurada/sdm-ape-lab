## 문제 해결

### ❌ Ctrl+Shift+A가 작동하지 않아요

**원인:**
- Chrome 확장 프로그램이 단축키를 가로챔
- 브라우저 설정 충돌

**해결책:**
1. ✅ **방법 3 사용**: `Ctrl+Shift+E` 시도
2. ✅ **방법 1 사용**: URL에 `?admin` 추가
3. ✅ **방법 4 사용**: 로고 트리플 클릭
4. Chrome 확장 프로그램 비활성화
5. 시크릿 모드에서 테스트

---

### ❌ 로고 트리플 클릭이 안 돼요

**원인:**
- 클릭 속도가 너무 느림 (1초 초과)
- 로고 영역을 정확히 클릭하지 않음

**해결책:**
1. 더 빠르게 클릭 (1초 안에 3번)
2. 로고 텍스트 영역을 정확히 클릭
3. ✅ **방법 1 사용**: URL에 `?admin` 추가

---

### ❌ 비밀번호 입력창이 표시되지 않아요

**원인:**
- JavaScript가 비활성화됨
- 브라우저 캐시 문제

**해결책:**
1. 강력 새로고침: `Ctrl+Shift+R` (Windows) / `Cmd+Shift+R` (Mac)
2. 시크릿 모드에서 테스트
3. 다른 브라우저 사용
